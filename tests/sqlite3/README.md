# SQLite3 Benchmarks

Download 2 zips from the [official website](https://www.sqlite.org/download.html)

- sqlite-amalgamation-[ver].zip
- sqlite-src-[ver].zip

From the amalgamation zip, copy `sqlite3.c` and `sqlite3.h` to the current folder.

From the src zip, copy `speedtest1.c` from the `test` folder.

Run `run.sh` to compile.
